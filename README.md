# Maze Canvas

An interactive web app where you draw a solution path and a maze generates around your cursor in real time.

**[Try it live](https://rjayasin.github.io/maze-canvas/)**

## How to use

1. Click and drag on the canvas to draw a path
2. A maze generates around your cursor as you draw
3. Use the brush size slider to control the maze generation radius
4. Click Clear to start over
